# Vietnamese Course Image Guide

This folder contains custom images for the Listen & Practice vocabulary cards.

## 📁 Folder Structure

```
images/
├── food/           # Food & dining items
├── transportation/ # Vehicles and transport
├── activities/     # Daily activities and actions
└── places/         # Locations like markets, hospitals, etc.
```

## 🖼️ Image Requirements

### Recommended Format
- **Format:** JPG or WebP
- **Dimensions:** 600-800px wide × 400-500px tall
- **Aspect ratio:** 16:10 or 4:3
- **File size:** Under 200KB (compress for faster loading)

### Naming Convention
Use simple, descriptive lowercase names matching the vocabulary:
- `pho.jpg` - Bowl of pho noodle soup
- `com.jpg` - Bowl of cooked rice
- `coffee.jpg` - Vietnamese coffee
- `bus.jpg` - Vietnamese bus
- `taxi.jpg` - Taxi/ride share vehicle
- `motorbike.jpg` - Motorbike (xe máy)
- `market.jpg` - Vietnamese market scene

## 🎨 Image Guidelines

### Style
- Use bright, clear, well-lit photos
- Show authentic Vietnamese contexts when possible
- Avoid cluttered backgrounds
- Focus on the main subject

### Cultural Considerations
- Show real Vietnamese food/places/vehicles when possible
- Vietnamese coffee with phin filter is more authentic than generic coffee
- Show local Vietnamese markets rather than Western supermarkets
- Use photos that respect Vietnamese culture and context

### Example Sources
- Personal photos from Vietnam
- Stock photo sites (Unsplash, Pexels, Pixabay)
- Creative Commons licensed images
- Original photography

## ⚙️ How It Works

The cards use **automatic fallback**:
1. First tries to load your custom image
2. If image not found → shows emoji icon instead
3. No broken images or errors shown to students

This means you can add images gradually - cards without images will still work!

## 🚀 Adding New Images

1. Take or download your photo
2. Resize to recommended dimensions (optional but recommended)
3. Compress if over 200KB
4. Name it according to the list above
5. Save it in the appropriate folder
6. Refresh the page - image should appear!

## 💡 Tips

- Start with the most important food items (pho, com, coffee)
- Transportation photos are very useful for visual learners
- Market photos showing Vietnamese culture are engaging
- You can update/replace images anytime without changing code
