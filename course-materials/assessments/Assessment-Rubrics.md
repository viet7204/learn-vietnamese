# Assessment Rubrics
## Vietnamese for Adult Beginners Course

## 1. Oral Presentation Rubric (Week 12 Final Project)

**Total Points: 20**

### Pronunciation & Tones (6 points)

| Level | Points | Description |
|-------|--------|-------------|
| **Excellent** | 5-6 | Tones are consistently accurate; pronunciation is clear and understandable; minimal errors that don't impede communication |
| **Good** | 3-4 | Tones are generally accurate with occasional errors; pronunciation is mostly clear; some errors but meaning is still conveyed |
| **Developing** | 1-2 | Frequent tone errors that sometimes impede understanding; pronunciation needs improvement; communicates basic ideas despite errors |
| **Needs Work** | 0 | Tone accuracy is lacking; pronunciation unclear; difficult to understand even with  context |

### Vocabulary & Grammar (5 points)

| Level | Points | Description |
|-------|--------|-------------|
| **Excellent** | 4-5 | Uses rich, varied vocabulary; grammatical structures are accurate and varied; minimal errors |
| **Good** | 3 | Uses adequate vocabulary; basic grammatical structures are used correctly; some errors that don't impede communication |
| **Developing** | 1-2 | Limited vocabulary; simple structures with frequent errors; meaning is conveyed with difficulty |
| **Needs Work** | 0 | Very limited vocabulary; many grammatical errors; unable to convey meaning effectively |

### Fluency & Confidence (4 points)

| Level | Points | Description |
|-------|--------|-------------|
| **Excellent** | 4 | Speaks smoothly with natural pauses; confident delivery; good eye contact; minimal reliance on notes |
| **Good** | 3 | Generally fluent with some hesitation; reasonably confident; some reliance on notes |
| **Developing** | 2 | Frequent pauses and hesitation; appears nervous; heavy reliance on notes |
| **Needs Work** | 0-1 | Very halting delivery; lacks confidence; reads entirely from notes |

### Content & Organization (3 points)

| Level | Points | Description |
|-------|--------|-------------|
| **Excellent** | 3 | Well-organized; all required elements included; engaging and interesting |
| **Good** | 2 | Organized; most required elements included; adequate content |
| **Developing** | 1 | Somewhat disorganized; missing some required elements; limited content |
| **Needs Work** | 0 | Poorly organized; significant elements missing; insufficient content |

### Time Management (2 points)

| Points | Description |
|--------|-------------|
| 2 | Presentation is 4-6 minutes (within range) |
| 1 | Presentation is 3-4 minutes or 6-7 minutes (slightly outside range) |
| 0 | Presentation is under 3 minutes or over 7 minutes |

**Total: _____ / 20**

## 2. Conversation Assessment Rubric (Week 12)

**Total Points: 20**

### Comprehension (6 points)

| Level | Points | Description |
|-------|--------|-------------|
| **Excellent** | 5-6 | Understands all questions without repetition; responds appropriately; can follow topic changes |
| **Good** | 3-4 | Understands most questions; occasionally needs repetition; generally follows conversation |
| **Developing** | 1-2 | Understanding is limited; frequently needs repetition; struggles to follow conversation |
| **Needs Work** | 0 | Little to no comprehension; cannot respond appropriately even with repetition |

### Response Quality (6 points)

| Level | Points | Description |
|-------|--------|-------------|
| **Excellent** | 5-6 | Responses are complete and appropriate; uses varied sentence structures; adds relevant details |
| **Good** | 3-4 | Responses are generally appropriate; uses basic structures correctly; adequate detail |
| **Developing** | 1-2 | Responses are brief or incomplete; limited structures; minimal detail |
| **Needs Work** | 0 | Unable to formulate appropriate responses |

### Pronunciation & Tones (4 points)

| Level | Points | Description |
|-------|--------|-------------|
| **Excellent** | 4 | Consistently accurate tones; clear pronunciation; easily understood |
| **Good** | 3 | Generally accurate tones; pronunciation is understandable despite some errors |
| **Developing** | 1-2 | Frequent tone/pronunciation errors that sometimes impede understanding |
| **Needs Work** | 0 | Poor tone/pronunciation; difficult to understand |

### Interactive Communication (4 points)

| Level | Points | Description |
|-------|--------|-------------|
| **Excellent** | 4 | Can ask questions; initiates topics; maintains conversation flow; repairs communication breakdowns |
| **Good** | 3 | Asks some questions; responds to topics; generally maintains flow |
| **Developing** | 1-2 | Rarely asks questions; struggles to maintain flow; difficulty repairing breakdowns |
| **Needs Work** | 0 | Cannot maintain conversation; no questions asked; relies entirely on teacher to carry conversation |

## 6. Homework & Weekly Quiz Rubric (Ongoing)

**Total Points: 25 (accumulated)**

### Weekly Quiz Performance (15 points total)

**Average of all weekly quizzes (typically 10-12 quizzes)**

- Convert quiz scores to percentage
- Percentage × 15 = points earned

**Example:** Average quiz score 85% → 0.85 × 15 = 12.75 points

### Homework Completion & Quality (10 points total)

Evaluated and accumulated throughout course:

| Level | Points per Assignment | Description |
|-------|----------------------|-------------|
| **Excellent** | 2 | Complete, thorough, on time; demonstrates effort and understanding |
| **Good** | 1.5 | Complete, on time; adequate effort |
| **Developing** | 1 | Incomplete or late; minimal effort |
| **Needs Work** | 0-0.5 | Missing, very late, or insufficient effort |

**Calculate:** Average homework score × 5 = final homework points

## Grading Scale

### Overall Course Grade

**Total Possible Points: 100**

- Class Participation & Engagement: 20 points
- Weekly Homework & Quizzes: 25 points
- Mid-term Assessment (Week 4): 15 points
- Progress Evaluation (Week 8): 15 points
- Final Project & Assessment (Week 12): 25 points

### Letter Grade Conversion

| Percentage | Letter Grade | Description |
|------------|--------------|-------------|
| 93-100% | A | Excellent - Exceeds expectations |
| 90-92% | A- | Excellent |
| 87-89% | B+ | Very Good |
| 83-86% | B | Good - Meets expectations |
| 80-82% | B- | Good |
| 77-79% | C+ | Satisfactory |
| 73-76% | C | Satisfactory - Some gaps |
| 70-72% | C- | Satisfactory |
| 67-69% | D+ | Needs Improvement |
| 63-66% | D | Needs Improvement |
| 60-62% | D- | Needs Improvement |
| Below 60% | F | Unsatisfactory |

**Note for Adult Learners:** This course emphasizes growth and effort. If you attend regularly, complete assignments, and participate actively, you will succeed!
